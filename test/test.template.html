Viewing as:

When previewing other timezones, you may see generic timezone offset descriptions like "GMT+1". The JS Intl class only provides short descriptions (e.g. "BST") for select timezones in each locale (e.g. "en-gb"). This is expected and prevents confusion when multiple timezones use the same abbreviation.
